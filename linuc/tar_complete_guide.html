バラバラのファイル
a.txt b.txt c.txt
→
tar でまとめる
📦
.tar ファイル
まとまってるだけ
サイズはほぼ同じ
→
gzip で圧縮
🗜
.tar.gz ファイル
まとめて＋小さい
よく見る形式
tar 単体 = まとめるだけ（圧縮しない）
tar + z = gzip 圧縮も一緒にやる（.tar.gz）
tar + j = bzip2 圧縮も一緒にやる（.tar.bz2）

オプションは「3種類」に分けて覚える

① 必須：何をするか（1つだけ選ぶ）
c
create（作成）
アーカイブを新しく作る。ファイルをまとめるとき。
x
extract（展開）
アーカイブを展開する。中身を取り出すとき。
t
list（一覧）
中身の一覧を表示する。展開せずに確認したいとき。
② 圧縮形式：どの圧縮を使うか（省略可）
z
gzip 連携
.tar.gz や .tgz のときに付ける。gzip と組み合わせる。
j
bzip2 連携
.tar.bz2 のときに付ける。bzip2 と組み合わせる。
③ 補助：よく一緒に使うオプション
v
verbose（表示）
処理中のファイル名を画面に表示する。確認用。
f
file（ファイル指定）
操作するアーカイブファイルを指定する。ほぼ必須。

試験に出る黄金パターン 3つ

作成 tar czvf archive.tar.gz ./dir
c = 作る z = gzip v = 表示 f = ファイル指定
dir フォルダをまとめて gzip 圧縮し archive.tar.gz を作る
展開 tar xzvf archive.tar.gz
x = 展開 z = gzip v = 表示 f = ファイル指定
archive.tar.gz を展開して中身を取り出す
確認 tar tzvf archive.tar.gz
t = 一覧表示 z = gzip v = 表示 f = ファイル指定
展開せずに中身のファイル一覧だけ確認する
